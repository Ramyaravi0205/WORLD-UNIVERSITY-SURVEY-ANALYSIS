# World University Survey Dashboard

## Overview

The **World University Survey Dashboard** is a Power BI project created to analyze and visualize university-related data from different countries and fields of study. This dashboard helps users understand student satisfaction, tuition trends, scholarship distribution, online class participation, and overall university ratings through interactive visualizations and KPI cards.

## Features

* Total student count and scholarship student analysis
* Average satisfaction score and tuition overview
* Campus facilities rating analysis
* Country-wise overall university rating comparison
* Program level distribution (Master, Bachelor, PhD)
* Field of study analysis with ratings and satisfaction scores
* Online class participation insights
* Interactive charts including bar charts, pie charts, waterfall charts, and treemaps

## Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Excel / CSV Dataset** – Data source preparation
* **DAX Measures** – KPI calculations and analysis
* **Data Cleaning & Transformation** – Power Query

## Key Insights

* Universities with better campus facilities tend to receive higher overall ratings.
* Scholarship students form a significant portion of the total student population.
* Satisfaction scores vary across different fields of study and program levels.
* Online class participation is higher in courses related to Computer Science and Social Sciences.
* Tuition fees and overall ratings differ between countries and academic programs.

## Dashboard Components

* KPI Cards
* Line Charts
* Bar Charts
* Pie/Donut Charts
* Waterfall Chart
* Treemap Visualization

## Purpose of the Project

The main objective of this project is to provide meaningful insights into university survey data and help users understand educational trends using data visualization techniques. This dashboard can support academic research, institutional analysis, and decision-making processes.
